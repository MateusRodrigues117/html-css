Sombras e Bordas

Principal

Sombras

Para colocarmos sombras nas caixas adicionamos a propriedade box-shadow para criarmos sombras, ela possue quatro valores a ordem é sempre essa:

  1. Deslocamento horizontal( h-offset )

    2. Quanto a sombra vai andar para a direita.( valores negativos causam deslocamento para esquerda )

  2. Deslocamento vertical( v-offset )

    3. Quanto a sombra vai andar para baixo. ( valores negativos causam deslocamento para cima )

  3. Embaçamento ( blur )

    4. Quanto a sombra vai se espalhar pelo fundo.

  4. Cor

    5. Indica a cor da sombra é possivel usar transparencia.

boc-shadow: 3px 5px 4px black;

As sombras geralmente sao pretas entao use a cor preta com variaçoes na transparencia.

Bordas

Para arredondar todos os vertices da borda usamos:

border-radius; 10px;

Tambem podemos indicar cada vertice individualmente:

border-radius: 10px 20px 30px 40px;

Assim como para margem podemos usar dois valores o primeiro indicando o canto superior esquerdo e o canto inferior direito o segundo para o canto inferior esquerdo e o superior direito.

border-radius: 10px 20px;
