Gradientes

Gradient linear:

Para criar um gradiente linear usamos a propriedade background-image:linear-gradient(to right,var(--cor3),transparent); e configuramos a direçao ( to right, to left, to top, to bottom ), a cor e se quiser a trasparencia.

Criamos um gradiente radial(circular) com a propriedade background-image:radial-gradient(circle,red,yellow,green); configuramos que é circular e suas cores

Atributos de Font

  • font-weight

    • Muda o peso da fonte colocando ou retirando o negrito dele.

  • font-style

    • Altera o estilo de um texto do normal para italico.

  • font-family

    • Parametro usado para mudar a familia da fonte.

  • font-size

    • Troca o tamanho da letra de um texto.

Shorthands

Se usa shorthands para simplificar as declaraçoe. Exemplo sem o uso de shorthand:

    p {
        font-family: Arial, Helvetica, sans-serif;
        font-size: 1em;
        font-style: italic;
        font-weight: bold;
    }

Agora o exemplo usando a shorthand para a propriedade font:

    p {
        font: italic bold 1em Arial, Helvetica, sans-serif;
    }

A ordem dos atributos de uma shorthand em CSS é importante. No caso da propriedade font, devemos informar na seguinte ordem:

  1. font-size
  2. font-variant
  3. font-weight
  4. font-size/line-height
  5. font-family

Alinhamento de texto

Existem quatro tipos de alinhamento de texto, centralizado a esquerda, a direita, justificado(se adapita a caixa ficando organizado) e no centro.

  • text-align: left;
  • text-align: right;
  • text-align: justify;
  • text-align: center;

Tipos

Glifos, letras, caracteres sao signos alfabeticos projetados para reproduçao de textos com as prenças mecanicas. Ex: A, B, C, D...

Familia tipografica

È o conjunto de glifos que possue as mesmas caracteristicas anatomicas independente das suas variaços.

Fontes

Sao o conjunto de glifos que formam uma familia tipografica.

Tamanho da fonte

O tamanho padrao da maioria das fontes é 16px, isso equivale a 1em. A parti dai podemos configurar o tamanho de um titulo como 2x o tamanho da fonte como 2em.

Medidas

As medidas de uma letra sao : cm ( centimetros ), in ( polegadas ), pt ( pontos ), pc ( paicas ), px ( pixels ), etc.
A recomendaçao da W3C é usar px e em.

Declaraçao

        h1 {
            font-family: 'Times New Roman', serif;
            font-size: 1.5em;
        }
