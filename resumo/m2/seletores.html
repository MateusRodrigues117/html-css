Principal

id / class

Um id vai identificar um elemento unico dentro da pagina atual, por isso so podemos ter apenas um elemento com um id especifico. Ja uma class vai identificar uma classe a qual um ou mias elementos perteçam, compartilhando caracteristicas em comum a todos o que façam parte do grupo.

No documento nao podemos ter dois ou mais elementos com o mesmo id mais podemos ter multiplos elementos com a mesma class.

Todo id em HTML é identificado nas CSS por um simbolo de #. Toda class em HTML é identificado nas CSS por um ponto.

Pseudo-classe

Pseudo-classe é uma palvra-chave adiconada as declaraçoes de um seletor apos um sinal de dois pontos ( : ) e especificam um estado especial de um elemento.

Existem varias pseudo-classes para estilos podemos citar: :hover, :visited, :active; :checked, :empty, e :focus.

Pseudo-elemento

Ja pseudo-elemento é uma palavra-chave adicionada as declaraçoes de um seletor apos dois sinais de dois pontos ( :: ) e permite que voce formate um pedaço especifico do elemento referenciado.

Os principais pseudo-elementos soa: ::before, ::after, ::first-letter, ::first-line.

Regras CSS

As regras sao colocadas no inicio do documento CSS para indicar algo.
Exemplo de regra: @charset"UTF-8" ( usada para gerar compatibilidade com codificaçao UTF-8 )
