Algumas Tags

  • < P >

    • Cria paragrafos.

  • < BR >

    • Faz quebra de linhas.

  • < IMG >

    • Responsavel por carregar imagens que estao na pasta do projeto ou de links externos.

  • < H1 >

    • Responsavel por marca os titulos e possue variaçoes que vao de <H1> ate <H6>.

  • < ADDRESS >

    • Tag usada para marca endereços.

  • < STRONG >

    • Usada para marca temos de grande importancia em frases.Logo eles apareceram em negrito.

  • < EM >

    • Marca termos que queremos dar enfase.Logo eles apareceram em italico

  • < MARK >

    • Delimita o texto que queremos marca como se estivessemos usando uma caneta marca texto.

  • < small >

    • Deixa o texto pequeno.

  • < DEL >

    • Usada para marca textos que podem ser lidos mais nao considerados.

  • < INS >

    • Sublinha o texto dizendo que ele é de grande importancia e que deve se prestar atençao nele.

  • < SUB >

    • Diminue o texto e o coloca em baixo como em H2O

  • < SUP >

    • Diminue o texto e o coloca em cima como em X2 + 102.

  • < CODE >

    • Usada para delimitar um codigo com a grande vantagem das letras estarem mono-espaçadas facilitando a leitura.

  • < PRE >

    • Mantem o texto pre-formatado da mesma maneira que foi digitado, mantendo as quebras de linha.Usar <code> dentro de <pre> melhora a apresentaçao do codigo.

  • < Q >

    • Usada para marca citaçoes, o texto escrito dentro ganha aspas mas nao o deslocamento.

  • < BLOCKQUOTE >

    • Cria citaçoes que ganham quebra de linha e deslocamento alem das aspas.Tambem permite refernciar o author colocando o link para o texto original dentro das aspas do parametro: cite="".

  • < ABBR >

    • Sempre que quiser usar uma sigla a coloque dentro de abbr e use o parametro: tilte="" para colocar seu significado dentro das aspas.

  • < BDO >

    • Faz o texto ser escrito de tras para frente usando o parametro: rtl="" se indica a direçao se vai ser da esqueda para a direita se usa o valor: ltr mas se for da direita para a esquerda se coloca o valor: rtl como no texto seguinte vermelho é a cor mais linda do mundo.
