Imagens

Para carregar imagens no site usamos a tag img, mas quando prescisamos criar imagens que se adaptam ao tamanho da tela temos que usar a tag picture que é responsavel por concentar todas as fontes de uma imagem, cada fonte contendo a imagem especifica para cada proporçao da tela.

As fontes sao colocadas dentro da tag source (source media="") que é inserida dentro da tag picture junto com a img contendo a imagem padrao.

Ordem

A ordem entre os source é importante, os itens mais acima tem que ser os de menores tamanhos para max-width e os seguintes sejam maiores, de forma crescente. O ultimo item dentro de picture deve ser a imagem padrao.

Assim como no EX:


<picture>
    <source media="(max-width:600px )" srcset="../imagens/elfoescritor.png">
    <img src="../imagens/espadavermelha-g.png" alt="">
</picture>

Parametros de source

  • type=""

    • Indica o media type da imagem.

  • srcset=""

    • Configura o local da imagem que aparecerar quando o tamanho for atingido.

  • media=""

    • Indica o tamanho que deve ser considerado para o carrregamento das imagens.
