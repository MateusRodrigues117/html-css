Links

Hyperlinks sao ligaçoes entre um ponto a outro na world wide web.Toda vez que cliclamos em um local que nos direciona para outra pagina estamos interagindo com hyperlinks.

Para criarmos um link, devemos criar ancoras usando a tag a que determina que o texto inserido nela se torna uma ligaçao para outro lugar na internet.EX:

Main theme ASC3

 
    Main theme <a href="URL" target="_blank" rel="external">ASC3 </a>

Parametros de a

  • href=""

    • Dentro de suas aspas se coloca a url.

  • hreflang=""

    • Permite indicar para o navegador o idioma original da pagina a qual o link esta redirecionando, basta colocalo entre as aspas.

Atributos de a

  • target=""

    • Indica a aba em que o link abrirar,possue os seguintes valores:

    • _blank

      • Abri o link em uma novas aba no mesmo navegador.

    • _self

      • Abri o link na aba atual do navegador.

  • rel=""

    • Indica qual é a natureza do link, seus valores:

    • next

      • Indica que o link é a proxima parte da pagina atual.

    • prev

      • Indica que o link é a parte anterior da pagina atual.

    • author

      • Indica que o link leva para pagina do author do artigo.

    • external

      • Indica que o link leva para uma pagina externa.

    • nofollow

      • Indica que é um link nao adoçado pelo author da pagina, geralmente usado para links de anuncios.

    • dofollow

      • È o oposto de nofollow.

Downloads

Para criarmos um link para efetuar o download de algum arquivo basta colocar o caminho do arquivo dentro de href="", adicionar o atributo download="" com o valor configurado para o nome do arquivo e usar o atributo type="" para indicar o tipo do arquivo (media types) que esta sendo baixado.

Linux

Se estiver sereferindo a pasta atual do servidor pode usar ./ antes do nome do arquivo.Se quiser se referir a pasta imediatamente superior na hierarquia, use ../ para voltar um nivel para a pastar mais externa.
