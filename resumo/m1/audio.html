Principal

Audios

Dentro da tag audio adicionamos varios source (source src="") com os formatos diferentes do mesmo audio.

Coloque na parte de cima o seu formato favorito, os demais so serao carregados caso o primeiro falhe. Caso todos falhem se cria um paragrafo para ser efetuado o download do arquivo.

Atributos de audio

  • preload=""

    • Indica se o audio sera pre carregado ou nao e aceita tres valores:

    • metadata :

      • Vai carregar somente as informaçoes sobre o aquivo.(tamanho,tempo,imformaçoes sobre direitos, etc)

    • none :

      • Nao vai carregar absolutamente nada ate o usuario apertar play ou um script inicie a reproduçao.

    • auto :

      • Vai carregar o arquivo de audio inteiro assim que a pagina for carregada, mesmo que o usuari nunca aperte play. (padrao)

  • controls :

    • Apresentarar o play na tela. Caso nao seja colocado na tag audio o player ficarar com uma cor transparente e o usuario nao poderar interagir.

  • autoplay :

    • Quando inserido vai iniciar a reproduçao do audio assim que a pagina for carregada.

  • loop :

    • Faz o audio ser reproduzido eternamente.

Types

  • mp3 / audio/mpeg
  • wav / audio/wav
  • ogg / audio/ogg
