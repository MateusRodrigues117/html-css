Formataçao de Texto

Principais Formataçoes

  • Destaque

    • Nessa frase, tenho um termo em destaque usando a tag < STRONG >.

  • Enfase

    • Nessa frase, temos um texto em enfase usando a tag < EM >.

  • Texto Marcado

    • Podemos tambem criar um texto marcado usando a tag < MARK >.

  • Texto Pequeno

    • Aqui temos um texto pequeno usando a tag < SMALL >.

  • Texto Deletado

    • Podemos marcar um texto como excluido para indicar que ele pode ser lido mais nao considerado usando a tag < DEL >.

  • Texto Inserido

    • Podemos marca um texto como inserido para indicar que ele foi adicionado depois usando a tag < INS >.

  • Texto Sobrescrito

    • Serve para inserir coisas do tipo x 20 + 3 usando a tag < SUP >.

  • Texto Subscrito

    • Serve para inserir coisas do tipo H2O usando a tag < SUB >.

  • Codigo-Fonte

    • O comando document.getElementById('teste') é escrito na liguagem javascript. Usando a tag < CODE > permite mostra-lo de maneira melhor.

  • Pre-formataçao

    • O comando 

    document.getElementById('teste')
    é escrito na liguagem javascript. Usando a tag < PRE > permite mostra a identaçao de um codigo e junto com a tag < CODE > o resultado é melhor.
