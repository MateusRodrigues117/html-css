Trabalhando com links

Tag <A>

Ela é usada na criaçao de links(ancoras) para isso basta colocar qualquer texto ou imagem dentro dela.Nessa tag em href="" dentro das aspas se coloca a URL de algo.

Parametros de <A>

  1. Target=""

    2. O parametro target="" possuie os seguintes atributos:

    • _blank /indica ao navegador que a pagina linkada sera aberta em uma aba em branco
    • _self /indica ao navegador que a pagina linkada sera aberta na mesma aba.

  2. Rel=""

    3. O parametro rel="" possue os seguintes atributos:

    • External /significa que é um link referente a uma pagina externa.
    • Next /diz que o endereço linkado é a proxima pagina.
    • Priev /diz que o endereço linkado é a pagina anterior.
    • Nofollow /faz referencia de que voce nao da aval ao site linkado.
    • Dofollow /Significa que voce da aval ao site linkado.

Para retorna para a pagina anterior se usa ../Nome da pasta.
Isso no caso de paginas locais como as desse caderno.

Dowloads

Para criar um link que vai gerar um download é so colocar o endereço do arquivo em href="" e usar os parametros download="nome" e type="application/pdf" para pdfs salvos local.
