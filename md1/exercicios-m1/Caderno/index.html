Representaçao de Dados

0 e 1(binary digit = bit)

8 bits é a porçao minima para se representar algum dado, 8 bits recebe o nome de byte.(um byte é igual a 8 bits)
EX:01000001 = A.

Multiplos de byte

  • 1024 bytes = 1KB (1 Kilo byte)
  • 1024 KB = 1MB ( 1 Mega byte)
  • 1024 MB = 1GB (1 Giga byte)
  • 1024 GB = 1TB ( 1 Tera byte)

Diferença entre MB e Mb

  • MB = Mega byte e é usado para armazenamento.

    • Armazenamento exemplo HDS

  • Mb = Mega bits e é usado para transmisao.

    • Transmisao exemplo Internet.

Dominio e Hospedagem

Dominio

Dominio é o nome que identifica o site, possue nome unico é pago anualmente e possuie varios TLDs.

Hospedagem

Hospedagem é o espaço para armazenamento de arquivos, é pago mensalmente e possue outros recursos.

URL

www.github.com/gustavoguanabara

  • www = sub-dominio.
  • github.com = dominio
  • .com = TLD
  • /gustavoguanabara = caminho

HTML / CSS / JS

  • HTML

    • È o conteudo de um site.(Textos, imagens, videos , tabelas etc.)

  • CSS

    • È o design de um site.(Cores, sombras, tamanhos, posicionamento etc.)

  • JS

    • È as interaçoes de um site.(Menus, animaçoes, popups, validaçoes etc.)

Front-end e Back-end

Front-end

È voltado para o lado visual e para interaçao direta do usuario.
Tambem chamado de client-side possue foco nas seguintes tecnologias: HTML, CSS e JS.

Back-end

Responsavel pelo armazenamento e manipulaçao de dados nao é diretamente acessado pelo usuario.
Tambem chamado de serve-side possue foco nas seguintes tecnologias: PHP, JS, C#, Ruby e Python
