Trabalhando com listas

Listas Ordenadas

Uma lista ordenada é criada com a tag <OL>, essa lista possue um indicador para cada item mostrando sua possiçao, geralmente começando do item 1.

Dentro de <OL> se coloca a tag <LI>, nessa tag se coloca o item da lista.
Exemplo de lista:

    Jogos de Stealth que eu joguei colocados na ordem em que joguei.

  1. Dishonored
  2. Metal Gear Solid V
  3. Dishonored death of the outsider
  4. Aragami 2
  5. Aragami

Parametros de OL

  • Type="" muda os caracteres que numeram a lista.(1 A a I i)
    1. Exemplo
    2. Exemplo
    3. Exemplo
  • Start="" muda o numero de inicio da lista se for mudado para o numero 4 a lista começarar aparti dele.
    1. Exemplo
    2. Exemplo
    3. Exemplo

Listas nao Ordenadas

Uma lista nao ordenada é criada usando a tag <UL>, essa lista nao possue indicador de posiçao.

Dentro de <UL> se coloca a tag <LI>, nessa tag se coloca o item da lista.
Exemplo de lista:

    Jogos Coop que joguei.

  • Bordelands 1
  • Bordelands 2
  • Bordelands the pre-sequel
  • Aragami
  • Aragami 2
  • Halo 3
  • Astronier

Parametro de UL

    Type="" é usado para trocar o pontinho que fica no começo de cada item da lista, as variaçoes sao as seguintes:

    • disc
    • circle
    • square

Listas de Definiçoes

Usado a tag <DL> se cria uma lista de feniçao, dentro de <DL> é colocado mais duas tags a <DT> que é usada para se colocar o termo e <DD> usada para dizer a definiçao do termo, essas duas tags nao sao colocadas dentro da outra e sim uma em cima da outra na ordem <DT> / <DD>
Exemplo:

HTML
Liguagem de marcaçao para a criaçao do conteudo de um site.
CSS
Liguagem de marcaçao para a criaçao do design de um site.
Javascript
Liguagem de programaçao para a criaçao de interatividade de um site.
